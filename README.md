# Proyecto PHP

## Descripción
Página web sencilla en PHP que muestra un mensaje de "Hola Mundo" junto con una imagen generada con IA.

## Prerrequisitos
- Instalar PHP
- Comando para instalar php: sudo apt install php
- Instalar Apache2
- Comando para instalar apache2: sudo apt install apache2

## Pasos para ejecutarlo
1. Clonar el repositorio con el comando:
   git clone https://github.com/Sergio-AGM/Proyecto_PHP.git
2. Mover los archivos a la carpeta del servidor:
   sudo mv Proyecto_PHP /var/www/html/
3. Iniciar Apache:
   sudo systemctl start apache2
4. Abrir el navegador y escribir:
   http://localhost/Proyecto_PHP/Index.php
